# Arabic Learning App for Dyslexic Students

A comprehensive mobile application designed to support Arabic language learning for students with dyslexia through visual aids, interactive activities, and personalized learning experiences. The app is powered by a robust backend server that manages authentication, user roles, learning content, AI features, progress tracking, notifications, and secure data storage.

## 🌟 Features Overview

### Core Learning System

- **Dyslexia-Friendly Design**: Visual aids, color coding, and tailored difficulty levels
- **Gradual Learning Paradigm**: Progressive learning from letters → words → sentences → stories, orchestrated by backend APIs that deliver structured lessons and track user advancement
- **Interactive Multisensory Activities**: Games and activities following each lesson, with backend endpoints evaluating user input and providing real-time feedback
- **Expert-Backed Approach**: Developed with insights from psychologists and dyslexia specialists, with backend support for storing and referencing expert recommendations

### Advanced Technology Integration

- **Text-to-Speech System**: Reading and auditory support for all content, enabled by backend integration with TTS services
- **Speech-to-Text Integration**: Voice interaction and pronunciation evaluation at the letter, word, and sentence level, processed and scored by backend controllers
- **Object Detection Module**: Real-life object interaction for vocabulary learning, with backend endpoints handling image uploads and AI-based recognition
- **AI Story Generation**: Personalized stories based on user preferences, generated by backend services using advanced language models
- **Story Summarization**: Extractive summarization to assess comprehension, powered by backend AI modules

## 👥 User Roles

### Independent Learner

- Individual registration and app usage, managed by backend authentication and user profile services
- Personal progress tracking, with backend storing detailed learning analytics
- Self-paced learning experience, with backend APIs adapting content delivery

### Guardian Account

- Manage up to 3 children under supervision, with backend logic for linked accounts
- Access detailed performance reports, generated and sent by backend PDF and email modules
- Monitor learning progress and achievements, with backend aggregating and analyzing data
- Receive automated PDF reports via email, scheduled and delivered by backend jobs

## 🔐 Authentication System

- Secure sign up and login functionality, using JWT-based authentication and encrypted password storage in the backend
- Role-based access control for learners, guardians, and admins, enforced by backend middleware
- Data protection and privacy compliance, with backend validation and secure session management

## 🎓 Learning Structure

### Progressive Learning Path

1. **Letter Recognition**: Visual and auditory letter introduction, with backend tracking attempts and accuracy
2. **Word Formation**: Building vocabulary with object detection, backend evaluating spoken and visual input
3. **Sentence Construction**: Grammar and structure learning, backend scoring pronunciation and sentence formation
4. **Story Creation**: Narrative skills development, with backend AI generating and summarizing stories

### Interactive Activities

- Multisensory games following each lesson, managed by backend game and exercise controllers
- Pronunciation evaluation at multiple levels, with backend speech-to-text and scoring logic
- Real-time feedback and correction, provided by backend endpoints
- Adaptive difficulty adjustment, with backend analyzing user performance and adjusting content

## 🤖 AI-Powered Features

### Story Generation

- Personalized story creation based on user preferences, using backend integration with large language models
- Interactive storytelling experience, with backend generating follow-up questions
- Comprehension questions and assessments, created and evaluated by backend services

### Object Detection

- Real-time object recognition, with backend handling image uploads and AI inference
- Vocabulary building through visual association, backend linking detected objects to learning modules
- Camera-based learning activities, with backend storing and processing user submissions

### Voice Technology

- Advanced Speech-to-Text for pronunciation practice, backend processing and evaluating audio
- Natural Text-to-Speech for reading support, backend generating audio for lessons and stories
- Pronunciation accuracy evaluation, backend scoring and tracking user progress
